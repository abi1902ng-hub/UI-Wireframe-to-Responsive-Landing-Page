# UI-Wireframe-to-Responsive-Landing-Page

A simple and responsive **Doraemon Fan Website** built using **HTML** and **CSS**. This project was created by first designing a custom wireframe and then implementing it into a functional website. It helped me understand the importance of planning a user interface before development and provided hands-on experience with modern web layout techniques.

---

## 📌 Project Overview

This website is a fan page dedicated to Doraemon. It includes:

- 🏠 Navigation Bar
- 🎥 Hero Section with Call-to-Action
- 📖 About Doraemon
- ✍️ About the Author
- 👥 Character Gallery
- 🔘 Call-to-Action Button
- 📄 Footer

The project follows a clean and organized layout inspired by the wireframe created before development.

---

## 🎨 Design Process

Before writing any code, I designed a wireframe to visualize the structure and layout of the website. Planning the design beforehand made the development process more organized and helped me clearly understand how each section would fit together.

The wireframe served as a blueprint throughout the project, allowing me to focus on implementing each section one at a time.

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Flexbox

---

## 📚 What I Learned

This project was a valuable learning experience and helped me strengthen my front-end development skills. Through building this website, I learned:

- Creating a website from a custom wireframe.
- Structuring web pages using semantic HTML.
- Using CSS Flexbox for layouts and alignment.
- Designing responsive and organized page sections.
- Creating navigation links that scroll to different sections.
- Styling buttons, cards, and images using CSS.
- Adding hover effects and transitions for better user interaction.
- Organizing HTML and CSS into a clean and maintainable structure.

Most importantly, I gained a better understanding of how a basic website is planned, designed, and built from scratch.

---

## 💡 Challenges Faced

During development, I encountered a few challenges, including:

- Understanding how Flexbox works with direct child elements.
- Positioning and aligning content correctly.
- Creating full-screen hero sections using background images.
- Styling reusable components while maintaining consistency.
- Debugging layout and spacing issues.

Overcoming these challenges improved my understanding of CSS and front-end development.
